## 📄 Descrição

Sistema desktop de gerenciamento de estoque de farmácia desenvolvido utilizando **C# com Windows Forms** e **MySQL**.

A aplicação permite realizar operações de **CRUD (Create, Read, Update, Delete)** para controle de medicamentos, organizando-os por categorias e facilitando a gestão de entrada e saída de produtos.

***

## 🎯 Objetivo

* Controlar o estoque de medicamentos
* Organizar produtos por categorias
* Registrar entrada e saída
* Simular um sistema real de farmácia

***

## ⚙️ Funcionalidades

* ✅ Cadastro de medicamentos
* ✅ Listagem de produtos
* ✅ Atualização de dados
* ✅ Remoção de medicamentos
* ✅ Categorização de produtos
* ✅ Controle básico de estoque

***

## 🗂️ Estrutura do CRUD

| Operação | Descrição             |
| -------- | --------------------- |
| Create   | Adiciona medicamentos |
| Read     | Exibe os produtos     |
| Update   | Atualiza dados        |
| Delete   | Remove medicamentos   |
